# 자연어 처리 트렌드 변화

- 자연어 처리 분야는 전통적으로 확률 통계 기법에 기반한 알고리즘들을 사용하였습니다.
- 그러나 전통적인 확률 통계 기법의 한계점으로 인해 최근에는 딥러닝 기법에 기반한 자연어 처리 기법이 일반적인 트렌드가 되었습니다.
